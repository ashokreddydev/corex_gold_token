## 4. Login Flow (What Happens)

1. **Enter Credentials**
   - Email: Admin@gmail.com
   - Password: Admin@2026

2. **System Validates**
   - Checks if email exists
   - Verifies password is correct
   - Confirms account is active

3. **Login Successful**
   - You receive a security token
   - Token stored for future actions
   - Redirected to admin dashboard

4. **If Login Fails**
   - Error message shown
   - Can try again or reset password
   - Too many attempts = temporary lock

## 10. Troubleshooting

**Can't Login?**
- Check email spelling
- Verify password is correct
- Ensure Caps Lock is off
- Try clearing browser cache

**Forgot Password?**
- Click "Forgot Password" link
- Verify your email
- Follow reset instructions
- Set new password

**Account Locked?**
- Too many failed attempts = temporary lock
- Wait 30 minutes and try again
- Contact support if issue persists

**Session Expired?**
- Automatically logged out after inactivity
- Login again with your credentials
- No data is lost
